## Episode 1 美丽的大和之国的历女

>不知哪天起，脑袋里就住进了几只奇怪的妖精……我到底是招惹了什么东西啊……

在某个高中的教室里。

幡桐小历正像平时那样听着老师的讲课。突然，从她的脑海深处传来了奇怪的声音，让她不禁双手抱头。

“这、这是什么……总感觉好像听到了奇怪的声音……”

“幡桐同学，怎么了吗？是不是身体不舒服？”

“咦？不、不是的……我没事……”

“就算是考试结束也不能就这样放松下来啊。当然，这话也不只是说给幡桐同学听的哦。”

学生们有气无力地回应了老师的话。

就在小历打算再次集中精神上课的时候，脑海中又响起了声音。

<font color = violet>“——你——能够——”</font>

“哇啊啊啊啊啊啊！？到底怎么回事！？”

“幡桐同学！请保持安静！”

“不、不好意思……”

因为在课上突然发出怪声而成为班上焦点的小历，只能满脸通红地缩在自己的座位上。

“呜呜……到底是发生了什么事……妖精小姐啊……到底想要说什么啊……”

小历无法听清楚脑中的这几个声音在说什么，只能听到像杂音一样的东西。

但是渐渐地，声音也开始渐渐清晰起来。

<font color = violet>“就赌上被人称为天才阴阳师的，吾之先祖……晴明大人的名号！”</font>

“到、到底在说谁啊……对了，干脆试试看和他们对话吧。”

小历就像是祈祷一样，十指紧扣，轻声向着脑内的声音说到。

“妖精小姐……听得见我在说什么吗？如果听的见的话请回复……”

<font color = gold>“哦哦，听得见听得见。”</font>

“嗯……不知道妖精小姐能不能听得见呢……”

虽然沙音瑠回答了她的问题，但是她说的话并未传到小历的耳中。

八云她们也注意到了联系上的小历身上的特殊情况——


<font color = violet>“嗯……看起来这边的对话似乎没法完全传给她的样子。”</font>

<font color = gold>“那这该怎么办才能救她啊？”</font>

和小历取得联系的众人陷入了不安之中。

因为她们这次和小历取得联系的目的，就是为了回避将来某个时刻，即将发生在小历身上的“事故”。

<font color = violet>“那么，看来只能让她看看这看似风平浪静的日常背后，将会有怎样的命运在等待着她了吧……”</font>

“咦？好奇怪……为什么突然觉得好困……”

<font color = gold>“喂！在上课的时候是不能睡觉的吧！”</font>

“我……为什么会……难道说，这也是……妖精小姐……的影响吗……”

小历突然感觉到阵阵困意，就这样趴在桌子上睡着了。

——等她恢复意识的时候，她发现自己来到了一个并非教室的场所。

“咦，这里，到底是哪里啊……？”

从周围的布置来看似乎是个会场的样子，周围放着各种箱子一样的物体。虽然对眼前的一切没有印象，但却总有一种既视感。

“这个……难道是那种问答节目的布景吗？”

正如小历所说，眼前看到的正是那些问答节目等能见到的答题者的席位。但是，眼前的这个地方，既没有答题者，也看不到出题者的身影。

周围空无一人，只有小历一个人站在这里。

“咦？为什么……我会看到我自己？……难道说，这一切都是梦吗？”

既然自己能够看到自己的话，肯定是做梦之类的吧。小历不再多想。

然而，就在下一刻，周围就燃起了大火，将眼前的另一个自己围了起来。

大火没有停息的意思，只是慢慢地燃烧着舞台的布景，向着舞台中央的小历慢慢接近。

“难道说，那个‘我’，就要这样死了吗……”

明明是发生在自己身上的事情，现在却像是他人一般旁观着。

当然，会有这样的反应也不奇怪。

因为眼前正被火焰灼烧的的，既是自己，又不是自己。

但是小历的心中却有种不可思议的感觉。

仿佛就像正在体验自己过去的经历一样。

## Episode 2 考试后的群雄们

>“战国问答拉力赛”……看起来很有趣的样子……要是能够成为和大家在毕业之前的回忆就好了啊……

——翌日。

也就是八云她们和小历接触一天之后。

即便小历已经放学回家，八云她们的对话还是没有停止，小历也只能将脑中的声音放在脑后。

第二天早上醒来的时候——

“早上好……”

<font color = gold>“早上好……才怪啊！看看现在都几点了！这不都迟到了吗！”</font>

“呜呜，怎么又听到这个声音了……”

就像小历所说，妖精小姐的声音，又突然能听到了。

睡眼惺忪的小历从床上起来，呆呆地望向旁边的始终。

“啊，真的耶……现在出发的话，应该还能赶上第一节课。”

<font color = cyan>“哦哦，竟然还能好好上学，厉害啊！”</font>

<font color = gold>“这有啥了不起的？学生去上学不是很正常吗？”</font>

<font color = violet>“听说高中生应考结束之后，因为可以自己选择是否去学校，所以来学校上课的人也会变少。”</font>

<font color = gold>“哦，还有这种事情啊。”</font>

“呜呜，好像又在讨论什么有的没的事情……”

即便是在准备出门的时候，八云她们的声音还是在小历的脑中回响着。

例如内衣的选择啦，吃的早饭啦，总之各种各样的东西都有讨论，不过详细的对话内容并未传入小历的耳中。

结果，因为被头脑中的对话吸引了注意力，小历最终出门的时间晚了不少。

“至少……至少要赶上第二节课……”

为了前往学校坐上电车的小历，当坐到座位上之后，就从书包里掏出了一本小说。

这就是她上学路上的消遣。

也就是在电车上阅读历史系的小说。

虽然从外面看来小历只是静静地在看着书，但是脑中却已经炸开了锅。


<font color = red>“这个，是日本历史上的武将对吧？”</font>

<font color = violet>“是呢。虽然确实是那些历史上的名将，不过因为被编成了读物，所以一些情节会有些许夸张的。”</font>

<font color = gold>“没想到你还挺懂的啊。”</font>

<font color = red>“而且她的房间里也放了很多和战国时代相关的东西，其实这孩子是个战国历史宅吧？”</font>

<font color = violet>“嗯，应该是这样。”</font>

“真是的……能不能安静一下……”

虽然小历很想沉心与书本之中，但是八云她们的七嘴八舌总是打乱她的心绪，结果不但一页都没看完，甚至还坐过了头，坐到了终点站。

就在小历打算回头换乘别的路线回去的时候，一张广告的海报吸引了她的注意力。

<font color = gold>“怎么突然停下来了？”</font>

<font color = red>“看起来是在看眼前的这幅海报。‘战国问答拉力赛，正式开赛！’……这是什么？”</font>

“咦……看起来很有意思啊……”

就在小历仔细地确认了海报上的内容，走出车展之后，来梦突然想起了什么事情一样，大声喊了出来。

<font color = cyan>“啊！我想起来了啊！！”</font>

<font color = gold>“搞什么啊！突然喊那么大声！”</font>

<font color = cyan>“这里，是很有名的升学学校啊！之前就对这个地方有点印象，现在终于想起来了。”</font>

<font color = violet>“也就是说，这孩子的头脑也很聪明是吧？”</font>

<font color = cyan>“不过……看起来没什么朋友的样子。”</font>

<font color = red>“为什么这么说？”</font>

<font color = cyan>“因为一看就能明白。因为她……也跟我有一些相似之处啊。”</font>

之后，在稀稀落落的学校里，小历上了一天的课程，很快就来到了放学时间。

这周的打扫工作似乎是由小历负责的。

“哦，来了来了！看吧，我就说她还回来学校的嘛！”

“还真是……小历一直都是这么认真啊。”

“咦……康子？大家……也是来上课的吗？”

小历往发出声音的地方看去，原来是三名女学生正挥着手走了过来。

（注：以下三名女学生的名字恶搞了日本战国时期的三好三人众，即三好长逸（みよし ながやす）、三好政康（みよし まさやす）与岩成友通（いわなり ともみち））

<font color = gold>“这些家伙是谁啊！”</font>

“咦？那个……她们是同班同学的三好三人组，分别是康子（やすこ）酱，唯（うい）酱以及友明（ともみ）酱……”

<font color = gold>“咦？现在听得见我们说的话了吗？”</font>

“……咦？为什么，我能这么清楚地听见妖精小姐的声音啊……？”

“一个人在那里自言自语啥呢？”

“嗯嗯，没什么事哦。请问有什么事吗？记得今天好像没在学校看见你们的样子……”

“我们有个计划想找你商量，还是直接见面说比较方便。来，看看这个！”

一边这么说着，唯拿出来的，正是之前在车站看到的那张广告的海报。

“这、这是……”

“就算是小历也应该知道这个东西吧。我们仨，打算报名参加这个比赛呢。”

“咦？真的吗？”

“而且，我们仨本来就很喜欢战国史相关的东西。绝对能够拿到个好成绩！”

“再加上小历的话，我们组可就是如虎添翼了啊，肯定能够晋级的！”

“是这样啊……咦！？”

“所以啊，我们现在就出发吧！”

“现，现在吗！？”

“是啊，反正大扫除什么的，现在来学校的人也少了，不干也没人管的啦。”

就这样，康子直接拉着小历的手将她带离了教室。

另外两人也像是心有灵犀一般把小历的书包等各类杂物一起带了出去。

“咦，咦咦！？”

“怎么了？难道你没兴趣吗？”

“倒……倒也不是这么说……”

“那就没问题啦！”

结果，小历就这样半推半就地参加了比赛。

她就这样在半强制的情况下，被同学带去了比赛的会场。

——即便她不知道，这将是会影响她的未来的一大分歧点。

## Episode 3 大和西大寺的战斗

>明明我还有点自信的……果然，最后还是不行吗……

——夕阳时分，大和西大寺站。

似乎是在车站前面设置了战国问答拉力赛的临时会场。小历就这样被三好三姐妹带到了这里。

“哦——已经开始了吗！”

会场已经人声鼎沸，中间可以看到穿着战国武将的服装的主持人。

周围摆满了摄像机，以及同行的工作人员。

“那，那不是电视台吗……”

“只要我们参赛的话，说不定能上镜哦！”

“各位——！我从工作人员那里问到了规则哦。必须拿着专门的印章才能参赛，如果我们要参加的话就得赶紧去排队拿啊！”

“那么，赶紧找一下在哪里排队吧——哦，好长的队伍啊！再不快点排的话可就拿不到了哦！”

“啊，等等我！”

小历跟着三姐妹排在后面。

之后，小历从三姐妹那里得知，在比赛中根据回答正确的题目数量能够获取到特典的印章。

“我还在想是什么奖励，结果就只是张盖着印章的卡片吗~”

“也不错啦。就当成是毕业前的纪念品咯。反正大家都能拿到一样的啦！”

“对对。而且我们这边可是还有小历帮忙的，肯定能打个大胜仗的！”

“那个，只靠我，真的可以做到吗……”

“稍微有点自信吧。你可是被称为‘历女’的幡桐小历啊！”

虽然三姐妹努力地鼓励着面露怯色的小历，但是她的不安似乎仍未消除，仍然抿着嘴唇。

“要是完全回答不上的话该怎么——”


“呐！”

就在小历刚想叹气的时候，突然，有人从背后抓住了她的肩膀，吓得她大叫起来。

“哇！？”

“啊，抱歉抱歉。不是故意要吓你的啦。不过真的是好久不见了啊——！！”

“咦……咦……？”

小历回头看去，她看到的是一个跟她年纪相仿的女孩子。


扑面而来的活力，以及清凉的服装。

还有那丝毫不畏惧私人空间接近对方的交流方式。

光是看到这样的她，就可以知道她是那种世间常常被人称作“元气角色”的人了。

“怎么回事？你是小历的朋友吗？”

小历仔细打量着眼前女孩子的样貌，但是不知道是不是对方的气场太耀眼了，反而让小历变得畏畏缩缩。

看到这样反应的小历，辣妹风格的女孩子一边笑着，一边微微低下了头。

“不好意思啊，好像认错人了。刚刚突然和你搭话对不起了！”

“啊，没什么的……”

“……看你这个印章，是要参加比赛的，对吧？”

“啊……那个……是的……”

“我可不会输给你哦！”

“咦……？”

面对这突如其来的宣战公告，正当小历还在不知所措的时候，那个女孩子就已经走开，回到了队伍里。

“到底是怎么回事啊，那个女孩子……”

“感觉是生面孔，应该不是我们学校的吧，大概。”

“话又说回来啊，等我们参加完问答比赛之后，想一想之后还要去哪里玩吧！”

女孩子们的兴趣转移是很快的，三姐妹立刻就转到下一个话题了。

<font color = gold>“真是个奇怪的女孩子啊。”</font>

<font color = purple>“嗯，虽然看起来是认错人了，不过，真的只是这样吗？”</font>

<font color = red>“难道有什么值得注意的地方吗？”</font>

<font color = purple>“……不，只是这么感觉而已。”</font>

头脑中的八云她们仍然在意刚才的那个女孩子的情况，但是，现在没有更多的情报支持了。

就在各自思索的时候，问答比赛开始了。

“就，就要在这里进行比赛吗……”

矗立在小历面前的，是曾经在游戏厅见过的问答游戏机。

答题比赛就是利用这些游戏机跟会场中的其他人连线对战，一决高下。

“太好了……要是跟人面对面答题的话，肯定会因为紧张过头按不了按键吧……”

小历松了一口气。

当主持人宣布开始之后，游戏画面上出现了对手的头像和名称。

出现在屏幕上的是——

“这，这不是刚刚那个女孩吗……”

她们的对手，正是刚刚和小历搭话的那个辣妹风格的女孩子。

在玩家名称一栏，写的则是“PN.松永久秀”。

“这，这孩子看来也很喜欢那些武将的样子呢。没想到会有人在笔名上取这样的名字啊……”

虽然小历因为遇上同好心中暗喜，不过答题环节的到来让她又重新绷紧了神经。

然后，第一道问题出现了。

答题方式是抢答。

虽然小历立刻就将手伸向了回答键，但是对手的速度更快。

“怎么会……而且还答对了……”


明明都作出了正确的回答，但是第一回合就被人占得先机，小历的心中就像被泼了一盆冷水般。

然而比赛可不会因为她的沮丧而止步，下一道问题出现在了屏幕上。

“这，这样的话……！”

小历只能强打精神继续迎战，然而，不管是那些不知道答案的问题，还是那些自己知道正确答案的题目，当小历还在脑子里思考判断的时候，就已经被对方抢答了。

这场比赛，正因为是抢答这一形式，所以仅仅只靠丰富的知识阅历也是无法取得胜利的。

结果——小历她们以大败收场。

就在比赛结束，小历前去找三姐妹的时候。

刚才那个署名“松永久秀”的女孩子出现了。

“啊……那个……谢，谢谢对战——”

“我说你啊，也太菜了吧。”

“咦……”

“就这水平？亏你还被称作‘历女’呢。难道这几年间就让你退步到这个地步了吗？你这个杂鱼！杂鱼！”

对面毒辣的话语，让小历直接呆在原地。

想不到任何能够反驳的话语，小历只能默默地低着头不发一语。

“小历……”

<font color = gold>“搞什么啊那个女人！要是她敢这么在我面前撒野，早该给她一巴掌了！”</font>

<font color = violet>“不过，现在的我们也无法出手。而且，那个女孩……”</font>

<font color = cyan>“总觉得有些在意……”</font>

<font color = violet>“不错，总觉得在哪里见过的样子……还是说是我的错觉吗……”</font>

虽然没有小历那么严重，但是只能在远处什么都帮不上忙的八云她们也同样不甘心。

就这样，正式开始的问答比赛初战。

就像曾经的第六次筒井城之战那样，以“松永久秀”的胜利而告终。




## Episode 4 筒井城旧址之战

>呜呜，总感觉梦里出现的那个女孩子，身材好像很娇小的样子啊……

问答比赛初战铩羽而归的小历回到了家中。

在比赛结束之后仍然未能从失败的阴影中走出，回到房间之后就直接趴在了床上。
