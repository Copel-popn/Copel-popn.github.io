# 彩绿（イロドリミドリ）

诞生于CHUNITHM的原创女子乐队企划。人设为Hisasi。讲述数名舞原音乐大学附属舞原高中的少女为了取得学分开始组织乐队练习并表演等一系列的故事。目前故事线已到第二年的阶段，根据CHUNITHM稼动的版本对应了不同时期的剧情。

目前已经出了两季剧情，已完结。后续更新未知。

目前故事的载体包含有声漫画，漫画连载以及net上的人物故事三部分。本网站将收录net上的人物故事部分。

乐队名“イロドリミドリ”在剧情中是芹菜和同伴们吃不同颜色的咖喱的时候突然灵光一闪想出来的主意（有声漫画第五话）。

根据制作人的笔记，该乐队名取材自日文的熟语“いろとりどり”（各种各样）以及“よりどりみどり”（任意选择）这两个词汇的组合，代表角色之间充满个性而又相映成趣。乐队名本身也可以代表色彩鲜艳的绿色植物的意思，对应七名角色名字中和日本春季的七种蔬菜相关的内容。

文中将以“彩绿”统称“イロドリミドリ”。

## 学园祭篇（CHUNITHM无印）

学校中有一个传闻：“在学园祭上做出了优异表演的团体，好像会得到学分”。

名为明坂芹菜的少女，决定为了这个传言放手一搏，和其他志同道合的少女们开始了组成乐队的故事……。
