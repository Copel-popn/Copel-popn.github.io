以乐器演奏类曲子为主的系列。作曲者多为同人乐队的相关者。角色一般多为音乐家或者是手上持有类似乐器的东西。

本世界观的人物以恶神·GOD和正方的人们之间的战斗作为主轴，不过大部分的故事其实都为独立成篇的故事。
